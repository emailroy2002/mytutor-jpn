マイチューターについて

マイチューターはどんな英会話スクールですか
無料のインターネット電話Skype 又は ZOOMを使った英会話スクールです。
特徴として、担任制を選べるマンツーマン、試験対策に熟知した最高クラスの講師のみでレッスンを提供しています。
「IELTS 」「 TOEFL 」「英検」「 TOEIC 」 本格的な試験対策が格安料金で受講できます
レッスン時間 月～金 15:00～24:55、土・日 11:00～24:00
1レッスン＝25分となります。
1日の受講数に制限がないので、短期集中が可能です。
Skype 又は ZOOM を使ったレッスンの他に、添削サービスを提供しています。
ミニテスト、時間管理システムなど、学習成果をサポートする「パーソナル・サポート・プログラム」が無料でご利用いただけます。また、学習進捗状況が1目でわかる「パーソナル・サポート・プログラム」を全講師が共有することで的確な指導を行っています。
複数の科目（例えばスピーキング、ライティングなど）を同時に受講することが可能です。Multi Account を利用して、最短で4技能を習得することができます。
英会話は初心者ですが大丈夫ですか
はい、たくさんの初心者がマイチューターで学ばれています。マイチューターでは初心者用に豊富なレッスンコースをご用意しています。また、初心者向けのレッスンを得意とした担任講師をご用意しますのでご安心ください。
マイチューターのサービスを受けるために必要なものは何ですか
音声通話ソフト Skype 又は ZOOM が使える通信環境が必要です。
PC、タブレット、スマートフォンでご利用いただけますが、ライティングクラスなど一部のレッスンではPCでの受講をお勧めしています。
PCでは、ヘッドセットとウェブカメラがあればより効果的な授業になります。
無料体験レッスンはありますか
はい、初めての方のために無料体験レッスンを２回用意しております。特にご指定がなければレベルチェック又はフリーレッスンのお時間とさせていただきます。レッスンにご希望があれば「講師への連絡」から直接講師へお知らせください。
講師はどのような人たちですか
大卒資格を有するフィリピン人講師を厳選採用しております。公認英語講師資格（LET）者を中心に、IELTS,TOEFL,TOEIC 英検など、英語検定試験対策に強い最高クラスの講師を採用しています。
担任制について
お客様のご要望があれば担任講師を1名選出します。（担任制はお客様の任意です。必ずしも担任を決める必要はありません）担任講師と週に1回（毎月受講回数16未満の方）又は週に2回（毎月受講回数16回以上の方）固定レッスン時間を確保することができます。（例：水曜日　20:00）固定レッスン時間の予約は事務局で行います。固定レッスンの予約は通常通りキャンセルも可能です。
担任制の詳細はこちら »
講師は選べますか
はい、お選び頂けます。マイページの画面からご希望の講師を選ぶことができます。
子供でも受講できますか
はい、マイチューターでは年齢制限を設けておらず、どなたでも受講できます。予め御両親から目的やご要望を伺い、お子様に合った講師とレッスン内容をご提供いたします。

マイチューターでは、原則レッスン料以外に費用はかかりません。無料オリジナル教材も豊富に取り揃えてありますのでご自由にお使い頂けます。

レッスン料支払い後、いつから予約ができますか（ポイント設定までの時間）

レッスンに関する質問

レッスンの時間帯と休校日を教えてください
【休校日】1月1～2、12月23、24、25、31
の他フィリピンの祝祭日で講師数の大幅な減少が見込まれる場合は休講となる場合があります。その際は事前にウェブサイトやお客様マイページでご案内します。
レッスン時間　月～金　15:00～24:55 、 土・日　11:00～24:00
レッスンコース（カリキュラム）について教えてください
マイチューターでは、生徒様の目的やレベルに合わせて多彩なレッスンコースをご用意しております。
無料体験で希望するレッスンを受講できますか
はい。無料体験でご希望のレッスンを受講できます。希望レッスン（例：IELTS Speaking）を「講師への連絡」から担当講師に英文でお伝えください。
尚、レッスンのご希望がない場合は、原則、フリーレッスン又はレベルチェックとなります。
教材はどのように入手できますか
レッスンコースによりオリジナル教材、市販教材を利用しています。
各レッスンコース概要から使用教材をご確認ください。
市販教材は書店、アマゾンなどのネット通販で購入ください。
マイページにアップしていないオリジナル教材やご不明な教材については、マイページ上部メニュー「カスタマーサポート」又は担当講師にお尋ねください。
レッスンコースを途中で変更できますか
はい。できます。レッスンコースを変更する場合、予約した講師へ「講師への連絡」を通じレッスン詳細をお知らせください。(例：TOEFL　Speaking Question Type 3 Test 3)
通常、講師は前回の受講履歴を確認して継続したレッスンを開始します。なので、レッスン内容を前回と変更する場合はできるだけ詳細に講師へ伝えてください。
レッスンの予約はどのようにすればいいですか
レッスンの予約はマイページから授業開始３０分前までに行うことができます。ご希望で担任講師との固定予約を事務局でお取りします。 同じ講師のご予約は1日2コマ（50分）まででお願いします。
同じ講師を同じ日に2コマ以上予約する場合は、「特別予約」として受け付けますが、特別予約はキャンセル（欠席）してもポイントが消化されます。（利用規約第4条10項）
「予約数制限」と「同じ講師2コマを超えて予約」についての詳細はこちら
レッスンをキャンセルするにはどうすればいいですか
レッスンのキャンセルは授業開始3時間前までにお願いしております。マイページ「予約表」にある「キャンセル」ボタンをクリックしてください。
3時間を過ぎますと「キャンセル」ボタンから「欠席」ボタンに変わります。「欠席」ボタンをクリックして講師へお知らせください。「欠席」はポイントが消化されますが、「欠席」ボタンをクリックしませんと、講師は受講者へ何度も連絡することになりますので、是非ご協力ください。
レッスン時間に遅れてしましました
レッスンが始まって10分以内であれば、授業時間は短くなりますが、通常通り授業が行えます。ただし、10分以上経過してもご連絡できない場合は、欠席とさせていただきます。
予約した先生と違う先生とのレッスンになりました。どうしてですか
講師の体調不良などにより、予約していた講師が授業を提供できない場合、代わりの講師をこちらで手配させていただくことがあります。申し訳ございませんが、代講は緊急の事態と想定されますので、先生の選択はこちらにお任せください。ご理解のほど宜しくお願いします。
代講を用意しない設定をご希望の際は、マイページ上部にあるカスタマーサポートからご指示ください。
担任講師は替えられますか
はい、替えられます。レッスンを計画的に進めるためにはなるべくコース（カリキュラム）途中で担任講師を替えるのはお勧めできませんが、カスタマーサポートまでお知らせください。

ライティング（IELTS、TOEFL等）レッスンの受講方法について
スカイプを使わず添削だけでも受けられますか。
（スカイプはZOOMでも代用できます。）

ライティングレッスンの受講方法は「スカイプレッスン」と「添削サービス」の2種類あります。「スカイプレッスン」は、ご予約した講師とスカイプを使ったレッスンになります。１つの問題（トピック）を同じ講師で完成まで受講してください。問題（トピック）を前回のレッスンで受け取り（講師に依頼する）、予習されることをお勧めします。ただし、初回はレッスン開始時に問題を受け取ります　レッスンでは解答をセンテンス毎にスカイプ・チャットボックスを使い添削していきます。レッスン（25分）中に終わらない場合は、次の（2回目）レッスンまでに文章を完成させて2回目のレッスン前に、ファイル（Word等）を担当講師のスカイプへ送信してください。2回目のレッスンでは解答（文章）全体の添削、アドバイスを受けて終了となります。通常、180語以上を求められるIELTS Task2, TOEFL Task1,2, 英検®1級などは2回（50分）必要になります。

一方、「添削サービス」はスカイプを使わず、受講者が作成した英作文を講師が添削して、受講者のメールとマイページへ返信するサービスになります。

添削くんをご利用の方が、スカイプ（ZOOM）でライティングコースを受講する場合、講師が同じ質問をご用意する可能性がございます。その際は、講師に指摘して別の質問を受講してください。（混乱を避けるために、添削くん又はスカイプレッスンどちらかを継続して受講することをお勧めします）

ライティングの受講は、テキストファイルの送受信、キーボードの利便性などから、なるべくPCのご利用をお勧めします。iPadをご利用の場合は、スカイプレッスンでは、MS-Wordのインストール、添削サービスでは、iOS11以上のバージョンが必要です。

添削サービスの詳細はこちらをご覧ください。


レッスン内容は毎回講師に伝える必要はありますか

無料体験を含む初回レッスンでは、希望レッスン（例:IELTS Speaking）をご予約後マイページにある「講師への連絡」から事前にお伝えください。
2回目からは事前連絡の必要はありません。直近受講歴を参考に各コースカリキュラムの順番で進めていきます。但し、直近受講歴と異なるレッスンコース（コース変更）を希望される場合は、「講師への連絡」から希望レッスンを事前にお伝えください。
また、直近受講歴と異なるレッスンコースを購入された場合も、希望レッスンを「講師への連絡」から事前にお伝えください。一度、受講頂くと、カリキュラムの順番で進めてまいります。マイページに表示されておりますコース名も最終受講コースが表示されます。


トラブルに関する質問
トラブルシューティング－まとめ

レッスン中トラブルがあった場合どうすればいいですか

レッスン中に回線障害やスカイプ（ZOOM）不具合など、レッスン継続に支障があるトラブルがございましたら、マイページにある「講師への連絡 チャット」で講師にご連絡ください。

「講師への連絡 チャット」が繋がらない場合や、講師では問題解消が難しい場合は、マイページにある「セブマネジャー Chat Support」にご連絡ください。

「講師への連絡 チャット」、「セブマネジャー Chat Support」で問題が解消できず、レッスンが不成立になった場合、ポイント返還の最終判断はマイチューター事務局となります。

マイページ-上部メニューバーにある「カスタマーサポート」からトラブルの内容を記入しメールを送信してください。トラブル内容を調査して「マイチューター事務局」から生徒様へ48時間以内に返信させて頂きます。

マイページにログインできません
メールアドレス、またはパスワードが間違っている可能性があります。マイページのログイン画面下の「パスワードを忘れの方」からパスワードの再発行をお願いします。
無料会員登録後に登録確認メールが送られてきません
お使いになられているメールサービスによっては、「マイチューター　カスタマーサポート」からのお知らせが誤ってスパム・迷惑フォルダに分類されてしまうことがあります。スパム・迷惑フォルダに分類されてしまっていないかご確認ください。
レッスンの時間になっても講師からコールがかかってきません

3分過ぎても講師からコールが来ない場合は、マイページにある「講師への連絡 チャット」からご連絡ください。

「講師への連絡 チャット」 が繋がらない場合は、マイページにある「セブマネジャー Chat Support」までご連絡ください。

講師からコールが来ない原因は様々ございますが、最も多い原因は、初めての講師の場合、講師から届くコンタクトリクエストを承認していないことがございます。
レッスン開始5分前にスカイプ（ZOOM）にサインインしてコンタクトリクエストを承認して、コールをお待ちください。
稀にではございますが、スカイプ（ZOOM）の一時的な不具合などが原因で、講師からのコールが受講者に届かない場合がございます。
また、受講者が誤って別のスカイプ（ZOOM）でコールをお待ちになっていることが原因であることもございます。

トラブルのためレッスンが受けられない場合はどのような扱いになりますか
トラブルの原因が講師側に起因する場合は、ポイントをお返しさせていただきます。
トラブルでレッスンが受けられなかった場合は、原因を調査いたしますのでマイページ「カスタマーサポート」からご連絡ください。
当社規定の調査の結果、講師側に問題が見当たらない場合や、利用規約に定める非常事態、不可抗力はポイント返還の対象外となりますのでご了承ください。

トラブルのためレッスン時間が短くなった場合どのような扱いになりますか
生徒様側に起因する問題でレッスン時間が短くなった場合は、レッスン完了扱いになります。講師側に起因する問題でレッスン時間が短くなった場合、申し訳ございませんが、目安として20分のレッスン時間が確保されました場合は、レッスン完了とさせていただきます。20分に満たない場合は、カスタマーサポート宛にメールでご連絡下さい。ポイントをお戻しさせていただきます。
当スクールでは、全講師が光回線を使ったセンターからレッスンを提供させていただくなど音質の環境には努力をしておりますが、稀に回線が乱れたり、ビデオ通話ソフト（スカイプ等）の不具合などの原因でお客様にご迷惑をお掛けすることがございます。
レッスンの開始が遅れた場合でも、できる限り時間を延長するなど最大限努力を致しますが、2５分を確保できない場合もございますことをご了承ください。
